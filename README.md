# 📝 **My Blog**

A simple blog built with **Flask**, displaying posts loaded from a JSON file.  
The project includes a homepage listing all posts and individual pages for each article, styled with custom HTML and CSS.

---

## 🚀 **Features**

- **Dynamic post loading** from `blog-data.txt`
- **Homepage** showing all blog posts
- **Individual post pages** via `/post/<id>`
- **Jinja2 templates** for rendering content
- **Custom styling** with a clean card-based layout

---

## 🎨 **Templates**

### **index.html**
Displays all posts with title, subtitle, and a link to read more.

### **post.html**
Shows the full content of a selected post.
